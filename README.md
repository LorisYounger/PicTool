# PicTool
一个简单的图片处理工具

# 简单介绍
PicTool是一个简单的图片处理工具
可以对图片进行简单的加工 例如转换成灰度,清空透明色,分析轮廓图等
但是并不能像PhotoShop那样作画等高级操作,只是个简单的小工具

# 如何使用
见 Release 中直接下载即可使用
或前往 [Steam](https://store.steampowered.com/app/1381380/PicTool/) 免费下载
